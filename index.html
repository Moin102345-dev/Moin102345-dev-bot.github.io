Moderation

/kick <user> [reason]

Kicks a user from the server. Requires the Kick role.

/ban <user> [reason]

Permanently bans a user. Requires the Ban role.

/clear <channel> <amount>

Deletes a specific number of messages in a channel. Role: Clear.

/clear2

Deletes all messages in the current channel. Role: Clear2.

/nuke

Completely resets a channel (same as clear2). Role: Nuke.

Tickets

/ticketsetup <category> [staff_roles]

Sets up the ticket system in a category. Admin only.

/ticket

Opens a new support ticket.

/close

Closes a ticket (ticket owner or staff).

/reopen

Reopens a closed ticket (staff only).

/delete

Permanently deletes a ticket (staff only).

Reaction Roles

/rr <channel> <message_id> <role> <emoji>

Adds a reaction role. Requires the RRManager role.

Anti-Link & Automod

/bypass <mention|id>

Allows a user to bypass link protection. Admin only.

/bypasslist

Displays all users with bypass permission. Admin only.

/bypassremove <id>

Removes a user from the bypass list. Admin only.

Shutdown & Updates

/shutdownsetup <channel>

Defines the channel for shutdown and update logs. Admin only.

Premium

/preis

Shows current premium prices.

/redeem <key>

Redeems a premium key.

Say & Announce

/say <channel> <message>

Sends a message as the bot. Role: Say.

/announce <channel> <message> <time>

Schedules an announcement. Role: Announce.

/announce_list

Lists all scheduled announcements. Admin only.

/announce_cancel <id>

Cancels a scheduled announcement. Admin only.

Other Commands

/server

Creates and updates server roles. Admin only.

/lang <de|en>

Sets the server language. Admin only.

Extended Moderation

/mute <user> <minutes> [reason]

Temporarily mutes a user. Role: Mute.

/unmute <user>

Removes a mute. Role: Unmute.

/warn <user> [reason]

Warns a user. Role: Warn.

/warnlist <user>

Displays all warnings. Role: WarnView.
